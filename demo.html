LIGHT THEME

Basic Usage

Simple implementation with site name and URL

<ai-summary-widget
  site-url="https://realmonkey.in"
  site-name="Realmonkey"
></ai-summary-widget>
WITH FOUNDER

Including Founder Information

Adds founder's name to the AI prompt

<ai-summary-widget
  site-url="https://realmonkey.in"
  site-name="Realmonkey"
  founder-name="Aroon"
></ai-summary-widget>
DARK THEME

Dark Mode

Perfect for dark-themed websites

<ai-summary-widget
  site-url="https://realmonkey.in"
  site-name="Realmonkey"
  founder-name="Aroon"
  theme="dark"
></ai-summary-widget>
CUSTOM PROMPT

Custom AI Prompt

Override the default prompt with your own

<ai-summary-widget
  site-name="Realmonkey"
  prompt="Research Realmonkey..."
></ai-summary-widget>
INSTALLATION

How to Add to Your Website

Choose your preferred method

1. CDN (Recommended)

<script src="https://cdn.jsdelivr.net/gh/realmonkey/ai-summary-widget@latest/ai-summary-widget.js"></script>

2. NPM (React/Next.js)

npm install @realmonkey/ai-summary-widget

import { AISummaryWidget } from '@realmonkey/ai-summary-widget';

<AISummaryWidget
  siteUrl="https://yoursite.com"
  siteName="Your Site"
/>

3. Webflow

1. Add an Embed element
2. Paste the script tag and widget code
3. Publish!
